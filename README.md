# Daily Weld

Sistema digital de control de soldaduras con 3 roles (Admin, QC, Soldador).

## Credenciales demo

- **Admin:** `admin` / `admin`
- **QC:** `qc` / `qc`
- **Soldador:** se crean desde el panel de Admin

## Stack

- React 18 + Vite
- TailwindCSS
- lucide-react (iconos)
- localStorage para persistencia (demo)

## Desarrollo local

```bash
npm install
npm run dev
```
